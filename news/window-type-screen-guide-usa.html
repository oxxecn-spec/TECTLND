American homes have several window styles – each requires a slightly different window mosquito screen. This guide matches the best screen type to double‑hung, sliding, casement, and awning windows.

Double‑Hung Windows

The most common US window. Options: half‑screen (covers lower sash only) or full‑screen. Retractable screens allow you to roll the screen up when not needed.

Sliding Windows

Fixed frame screens attached to the stationary side, or retractable screens that pull across.

Casement & Awning Windows

These crank outward. Adhesive velcro screens are best – they don’t interfere with the mechanism and can be removed seasonally.

Window Sizes & Custom Orders

Most US windows are non‑standard sizes. MBORMC makes custom screens to fit exactly – no gaps.
